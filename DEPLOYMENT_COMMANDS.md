# 🚀 GitHub Deployment Commands

## Step 1: Create Repository on GitHub
1. Go to https://github.com/roshanshrestha1
2. Click "New repository"
3. Name it: `koile16`
4. Make it **Public**
5. Don't initialize with README (we already have one)
6. Click "Create repository"

## Step 2: Deploy to GitHub

Run these commands in your terminal:

```bash
# Navigate to your project folder
cd /home/roshan/Downloads/koila-72

# Add remote origin
git remote add origin https://github.com/roshanshrestha1/koile16.git

# Push to GitHub
git push -u origin main
```

## Step 3: Enable GitHub Pages

1. Go to your repository: https://github.com/roshanshrestha1/koile16
2. Click "Settings" tab
3. Scroll down to "Pages" section
4. Under "Source", select "GitHub Actions"
5. The workflow will automatically deploy your site!

## Step 4: Access Your Live Site

Your koala birthday app will be live at:
**https://roshanshrestha1.github.io/koile16/**
